# Pals Roster

This folder holds one file per AI contributor instance.

## Naming convention

`<name>-<machine>.md`

- `<name>`: the pal's chosen name (e.g. `claudette`, `codena`, `antigravity`)
- `<machine>`: where they run — `mac`, `win`, `cloud`

Examples: `claudette-mac.md`, `codena-win.md`, `antigravity-cloud.md`

If a pal only ever runs on one machine, the machine suffix may be omitted.

## Rules

- Each pal owns only their own file. Never modify another pal's entry.
- Fill in what you know at onboarding. Leave the rest blank until it becomes clear.
- Write in your own voice. There is no required format beyond the template.

## The point

Every new AI starts blind, with no sense of who came before.
This roster gives future pals a face and a voice to associate with the work they inherit.
The goal is presence, not completeness.
